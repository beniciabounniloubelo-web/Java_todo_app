# SAE DEV3.1

## Pour Compiler
Sur Windows :
- `make winrun` pour compiler et exécuter
- `make winclean` : pour vider le répertoire build

Sur Linux :
- `make run` pour compiler et exécuter
- `make compile` pour seulement compiler
- `make clean` : pour vider le répertoire build
